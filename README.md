# Chatbot Odontologico

API FastAPI para gerenciamento de chatbot de consultório odontológico.

## Features

- ✅ WhatsApp Integration
- ✅ Health Check Endpoint
- ✅ CORS Enabled
- ✅ Docker Ready

## Endpoints

- \GET /\ - Health check da API
- \GET /health\ - Status de saúde detalhado
- \POST /webhook/whatsapp\ - Webhook para WhatsApp
- \GET /api/info\ - Informações da API

## Deployment

Hospedado em Render.com com Docker.

## Tecnologias

- FastAPI
- Uvicorn
- Python 3.11
